Dashboard Help Guide

Getting Started

This dashboard provides insights into healthcare costs and insurance claims. Use the navigation menu on the left to explore different sections.

Main Features

  • Overview: See key metrics and high-level charts of healthcare expenses
  • Claims Analysis: Analyze claim data by procedure and medical condition
  • Cost Drivers: Identify factors driving healthcare costs
  • Payer Analysis: Compare insurance payers' coverage and performance

Using Filters

Use filters on the right panel to narrow down data by:

  • Demographics (age, gender, race)
  • Health conditions
  • Location
  • Time period

After selecting filters, click "Apply Filters" to update the dashboard.

Interacting with Charts

Hover over chart elements to see detailed values. Click on legend items to hide/show data series.

Exporting Data

Click the "Export Data" button to download the current view's data as CSV, Excel, or PDF format.
